我们生活在一个把「即时」当作默认价值的环境里。可真正耐看的体验，往往来自**延迟**：页面切换时那一点点缩放与淡入，卡片悬停时那一点点光晕与抬升。

延迟不是拖沓，而是给用户的大脑留出「预期」的空间。预期被满足的瞬间，会产生微小的奖赏回路——这就是质感。

在工程上，这意味着：

1. 为过渡选择更自然的缓动曲线（例如 `cubic-bezier(0.22, 1, 0.36, 1)`）。
2. 把阴影与发光当作「环境光」，而不是贴纸。
3. 尊重 `prefers-reduced-motion`，让美不成为负担。

```css
.surface {
  transition:
    transform 420ms cubic-bezier(0.22, 1, 0.36, 1),
    box-shadow 520ms cubic-bezier(0.22, 1, 0.36, 1);
}
```

当你愿意把 200ms 的差值当作设计变量来讨论，你就从「实现功能」跨进了「塑造气质」。气质无法被度量，但会被记住。
